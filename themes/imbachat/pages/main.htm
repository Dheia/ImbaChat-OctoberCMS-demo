Инструкция по интеграции чата

Установка плагина

Для начала установите плагин. Вы можете скачать плагин с нашего репозитория, и положить его в папку с таким путем "/plugins/imbasynergy/imbachatwidget". Либо в менеджере плагинов, в админки вашего сайта, найдите наш плагин через такое имя "ImbaChat-OctoberCMS"

Создание виджета

Теперь вам нужно зарегистрироваться. После этого вы попадете в панель администратора, где будет возможность создать виджет, путем заполнения полей, нажав на кнопку "Создать".

Настройка плагина в панеле администратора

Зайдите в панель администратора и нажмите "Settings", за тем нажмите на плагин "ImbaChat-OctoberCMS" и заполнить следующие поля:

  • dev_id - id виджета.
  • dev_login - нужно взять из поля "Developer login"
  • dev_password - нужно взять из поля "Developer password"
  • in_password - нужно взять из поля "Secret key"
